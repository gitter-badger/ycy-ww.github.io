hexo Nunjucks Errors 解决方法

Code：

原文：

文章里出现了Site updated: {{ now('YYYY-MM-DD HH:mm:ss') }} 的内容。然后 hexo 的文章渲染使用的是 Nunjucks ，他会将那几个大括号识别成自己的语法，这样生成的时候就会报错。Unable to call now

解决方法：

前后加一个这个： {% raw %} {% endraw %}
