辅助函数（Helpers）

辅助函数帮助您在模版中快速插入内容。辅助函数不能在源文件中使用。

网址

url_for

在路径前加上根路径，从 Hexo 2.7 开始您应该使用此函数而不是 config.root + path。

参数 描述 默认值
relative 是否输出相对链接 config.relative_link 的值

是否输出相对链接，默认遵循配置文件中 relative_link 的值

relative_url

取得与 from 相对的 to 路径。

示例：

gravatar

根据邮箱地址返回 Gravatar 头像 URL。

如果你不指定 options 参数，将会应用默认参数。否则，你可以将其设置为一个数字，这个数字将会作为 Gravatar 的大小参数。最后，如果你设置它一个对象，它将会被转换为 Gravatar 的一个查询字符串参数。

参数 描述 默认值
s 图片大小 80
d 默认头像
f 强制使用默认图象
r 头像等级限制

link_to

插入链接。

参数 描述 默认值
external 在新视窗打开链接 false
class Class 名称
id ID

示例：

mail_to

插入电子邮箱链接。

参数 描述
class Class 名称
id ID
subject 邮件主题
cc 抄送（CC）
bcc 密送（BCC）
body 邮件内容

示例：

image_tag

插入图片。

参数 描述
alt 图片的替代文字
class Class 名称
id ID
width 图片宽度
height 图片高度

favicon_tag

插入 favicon。

feed_tag

插入 feed 链接。

参数 描述 默认值
title Feed 标题
